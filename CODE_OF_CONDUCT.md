# Code of Conduct

## Our Commitment

We are committed to providing a welcoming, respectful, and harassment-free
experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus on technical issues, not people.
- Accept feedback gracefully.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or derogatory comments.
- Publishing private information without consent.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, and pull
requests that violate this Code of Conduct.

## Reporting

Please report incidents via the security contact process in `SECURITY.md`.
