# Support

## Usage Help

- Read `README.md` for common workflows.
- Read `docs/COMMANDS.md` for complete command reference.
- Use `pixshift --help` and command-specific `--help`.
- Review `docs/JSON_OUTPUT.md` for automation usage.

## Bug Reports

Please use the Bug Report issue template and include:

- exact command,
- sample input,
- output logs,
- OS + Python versions.

## Feature Requests

Please use the Feature Request issue template and describe:

- problem statement,
- expected command UX,
- expected output.
